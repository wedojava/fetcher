美国上周再次针对中国电信业龙头企业华为实施新的管制措施后，华为戏称，在美国的巨大压力下“还活着”。在美国步步进逼下，华为还能活多久？美国针对华为采许“申请许可制”，又真能堵住现有的法律漏洞吗？<br/> <br/> 华为5月18日开始在深圳召开第17届分析师大会（HAS 2020）。在一连四天活动的前夕，美国商务部宣布要修改《出口管理条例》（EAR），使用美国技术或设备的外国半导体业者，未来若要供货给华为及华为子公司海思半导体（HiSilicon）的海外企业，都要先向美国申请许可。<br/> <br/> 与此同时美国商务部第五度延长美国企业可以出货给华为的临时许可九十天，今年8月13日到期，预计这次延期将是“最后一次”。  
   
   
在深圳，没有炫丽的舞台、灯光设计，也看不到华为创办人任正非出席，华为刻意通过官网上的直播，让外界看到轮值董事长郭平摆出的低姿态，讲述去年在美国巨大压力下，华为这架“弹痕累累的老飞机”还在空中飞翔。<br/> <br/> “我们公司任总常说，华为就像这架千疮白孔的飞机，过去一年，补洞是我们的主旋律，也让我们皮糙肉厚了。好消息是，我们现在还活着。 ”<br/> <br/> 在美国一轮又一轮的施压下，华为活得如何？郭平说，华为去年营收达人民币8588亿元，净利则达到人民币627亿元。<br/> <br/> 另一方面，在美国五度延长临时许可期间，华为会采取双保险措施。<br/> <br/> 郭平说，华为去年一方面扩大投入研发资金达人民币1317亿元，增长近3成；另一方面扩大采购强化库存，金额达人民币1674亿元，增长逾7成。<br/> <br/> 华盛顿智库传统基金会亚洲研究中心专研经济与科技的资深政策分析师沃尔特斯（Riley Walters）告诉自由亚洲电台，风口浪尖上的华为短期会受伤，但在5G设备发展上，美国的做法只能拖慢华为，但不能致华为于死地。<br/> <br/> “我想华为是可以活下来的，原因有二。一是华为在美国之前不断展延临时许可期间扩大采购，强化库存；二是在美国压力下，这会警醒中国，一定要以举国之力扶植自己的半导体产业。”<br/> <br/> 沃尔特斯还说，美国要加强打压华为，全世界半导体产业也不会好过，因为对许多国家与厂商来说，华为是最大的芯片消费者。在美国的国家安全与企业的盈利上要如何取得平衡？他认为，美国与中国之间的科技战，接下来还会经历一段时间角力。 <br/> <br/> 他也指出，美国的单打独斗，效果也有限。面对华为，各个国家也有不同关切与考量，像是欧洲联盟就仍没有共识。  <br/> <br/> 《日本经济新闻》引述知情人士报道，华为与总部位于瑞士的义法半导体公司（STMicroelectronics）合作，开发旗下智能手机的芯片。双方从2019年已开始联合开发，但至今尚未公布结果。
